Oracle Client

Free, light-weight and easily installed Oracle Database tools, libraries and SDKs for building and connecting client applications to local or remote Oracle Databases

Oracle Instant Client enables development and production deployment of Oracle Database applications. It is used for popular languages and environments including Node.js, Python and PHP, as well as providing access for OCI, OCCI, JDBC-OCI, ODBC and Pro*C applications. ISVs and Partners can benefit greatly from being able to bundle Instant Client in their applications.
